# Example campaign package structure

This folder documents the zip layout produced by **Dispatch to AI Creative Team**.

```
campaign-package/
├── README.md
├── campaign.json
├── company-brain/
│   ├── brand-voice.md
│   ├── messaging.md
│   ├── personas.md
│   ├── positioning.md
│   └── products.md
├── campaign/
│   ├── campaign-summary.md
│   ├── article.md
│   ├── social-copy.md
│   ├── seo-strategy.md
│   └── visual-brief.md
└── creative/
    ├── required-assets.json
    ├── asset-checklist.md
    └── canva-template.json
```

Values are generated from the active organization's campaign context at dispatch time.

See `README.md` in this folder for a sample README body.
